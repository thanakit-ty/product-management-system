# Product Management System (Full-Stack)

A full-stack web application for managing products and categories. Built with **Vue.js 3** (Frontend) and **Node.js/Express** (Backend) with **MongoDB** database.

## üöÄ Features

- **Product Management:** Create, Read, Update, Delete (CRUD) products.
- **Category Management:** Create, Edit, Delete categories.
- **Search & Pagination:** Filter products with Regex search and browse via pagination.
- **Relational Data:** Products are linked to Categories using MongoDB references.
- **Responsive UI:** Styled with Tailwind CSS.

## üõ† Tech Stack

**Frontend:**
- Vue.js 3 (Composition API)
- Vue Router
- Axios
- Tailwind CSS (via CDN)

**Backend:**
- Node.js
- Express.js
- MongoDB (Mongoose)
- Dotenv
